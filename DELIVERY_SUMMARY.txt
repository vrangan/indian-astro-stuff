4️⃣  CONFIGURABLE SEARCH CRITERIA
   • SearchConfig dataclass for flexible event definitions
   • No code changes needed to modify criteria
   • Supports any combination of:
     - Sun position (Rasi and degree range)
     - Moon phase (Tithi window with tolerance)
     - Nakshatra requirements
     - Planetary aspects
     - Venus-Jupiter alignment options
     - Custom offset windows
   
   Example:
   ```python
   config = SearchConfig(
       sun_rasi_range=(270.0, 300.0),  # Makara
       target_tithi=14,                # Full Moon
       tithi_window=1,                 # ±1 tithi
       require_vj_alignment=True,      # Venus-Jupiter
       vj_tolerance_degrees=5.0,       # Within 5 degrees
       offset_window_days=15           # Skip days after match
   )
   ```

Date Column:

  Format: "YYYY BC/AD-MM-DD"
  
Positional Columns (0-360 degrees):
  • Sun_Longitude: Solar position
  • Moon_Longitude: Lunar position
  • Venus_Longitude: Venus position
  • Jupiter_Longitude: Jupiter position
  • Mars_Longitude: Mars position
  • Mercury_Longitude: Mercury position
  • Saturn_Longitude: Saturn position
  • JD: Julian Day number (for precise calculations)

Zodiacal Sign Columns:
  • Sun_Rasi: Solar zodiacal sign (e.g., "Makara (Cp)")
  • Moon_Rasi: Lunar zodiacal sign
  • Venus_Rasi: Venus zodiacal sign
  • Jupiter_Rasi: Jupiter zodiacal sign

Lunar Mansion Columns:
  • Sun_Nakshatra: Solar lunar mansion (27 options)
  • Moon_Nakshatra: Lunar lunar mansion (27 options)

Lunar Day Columns:
  • Tithi: Lunar day name (e.g., "Purnima")
  • Tithi_Index: Numeric value (0-29, 14=Full Moon, 29=New Moon)
  • Tithi_Progress: Percentage through current tithi

Planetary Relationship Columns:
  • VJ_Separation: Distance between Venus and Jupiter (degrees)

⚙️  CUSTOMIZATION EXAMPLES

Example 1: Search for New Moon Events
```python
custom_config = SearchConfig(
    sun_rasi_range=(270.0, 300.0),
    target_tithi=29,                # Amavasya (New Moon)
    tithi_window=1,
    offset_window_days=20
)
```

Example 2: Search for Mars-Venus Conjunction
```python
custom_config = SearchConfig(
    sun_rasi_range=None,            # Any sun position
    target_tithi=None,              # Any tithi
    require_vj_alignment=False,     # Disable V-J requirement
    required_aspects={
        'Mars': ['Venus']           # Require Mars-Venus
    }
)
```

Example 3: Search for Summer Solstice Events
```python
custom_config = SearchConfig(
    sun_rasi_range=(60.0, 90.0),   # Mithuna Rasi
    target_tithi=14,                # Full Moon preferred
    tithi_window=2
)
```
