USAGE WORKFLOW
==============

1. READ (15 minutes):
   └─ README.md → QUICKSTART.md

2. SETUP (5 minutes):
   $ pip install -r requirements.txt

3. RUN (5-30 minutes depending on date range):
   $ python ThiruppavaiDating.py
   Output: results_-6000_2100_enhanced.csv

4. VISUALIZE (1-2 minutes):
   $ python visualize_event.py results_*.csv 5
   Output: visualization_<date>.png (5 files)

5. ANALYZE (Varies):
   ├─ Open CSV in spreadsheet
   ├─ Review planetary positions
   ├─ Check visualization images
   └─ Compare with historical records

6. CUSTOMIZE (Optional):
   ├─ Edit SearchConfig in ThiruppavaiDating.py
   ├─ Change location (ACTIVE_LOCATION)
   ├─ Modify date range (START_YEAR, END_YEAR)
   └─ Adjust search criteria
   
   Then: Re-run scanner with new configuration

TROUBLESHOOTING REFERENCE
=========================

Issue: "Ephemeris files not found"
→ Solution: Verify de441_part-*.bsp files present

Issue: "ModuleNotFoundError"
→ Solution: pip install -r requirements.txt

Issue: No results found
→ Solution: Try require_vj_alignment=False

Issue: Slow performance
→ Solution: Use smaller date range for testing

Issue: Memory error
→ Solution: Increase offset_window_days (skip more days)

For details: See QUICKSTART.md troubleshooting section
